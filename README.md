# LuisFacunde.github.io

Este repositório contém os arquivos do meu website pessoal e portfólio profissional, hospedado diretamente no **GitHub Pages**.

O objetivo deste projeto é apresentar meu perfil, projetos acadêmicos e profissionais, além de servir como um espaço para demonstração de habilidades em desenvolvimento web.

---

## Tecnologias Utilizadas

O projeto é desenvolvido com foco em simplicidade, leveza e semântica moderna:

- **HTML5**: Estruturação semântica de todo o conteúdo das páginas.
- **CSS3**: Estilização visual, garantindo um design limpo e responsivo.

---

## Estrutura do Repositório

Atualmente, a estrutura básica do projeto na branch `main` consiste em:

```text
├── index.html       # Página principal do website (ponto de entrada)
└── README.md        # Documentação explicativa do repositório
```

## Como Acessar o Site

O site está publicado e pode ser acessado diretamente através do link:
**[luisfacunde.github.io](https://luisfacunde.github.io/)**

---

## Como Executar Localmente

Caso queira visualizar ou testar o projeto no seu computador local:

1. **Clonar o repositório:**
    ```bash
    git clone https://github.com/LuisFacunde/LuisFacunde.github.io.git
    ```
2. **Navegar até a pasta do projeto:**
    ```bash
    cd LuisFacunde.github.io
    ```
3. **Abrir no Navegador:**
    - Basta dar um duplo clique no arquivo `index.html` ou abri-lo diretamente com o navegador de sua preferência.
